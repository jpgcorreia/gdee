========================================
  Gene Discovery and Enzyme Engineering
========================================
**GDEE** is a Python package that provides the functionality necessary to run the Gene Disovery and Enzyme Engineering Platform developed under the project ShikiFactory 100 (European Union's Horizon 2020 research and innovation programme under grant agreement number 814408).

This code is available under the 

Requirements
============
* Python version 3+
* Numpy
* MDAnalysis
* MPI4Py

Optional requirements
=====================

Instalation
===========
Distributed packages
--------------------
1. Download the latest release_.
2. Run :code:`pip install gdee.version.tar.gz`

From source
-----------
1. Clone this repository
2. Run :code:`pip install miga/package`

Usage
=====
Plese refer to the examples_ folder and to `online documentation`_ to learn how to use this package.

Bugs and feature requests
=========================
Please report bugs and feature requests through the `Issues page`_.
